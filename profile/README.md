### Turing Pi Ecosystem

A full IaC stack for Turing Pi ARM clusters:

- **[terraform-provider-turingpi](https://github.com/freed-dev-llc/terraform-provider-turingpi)** — Terraform provider for BMC management
- **[terraform-turingpi-modules](https://github.com/freed-dev-llc/terraform-turingpi-modules)** — Reusable Terraform modules for cluster deployment
- **[turing-ansible-cluster](https://github.com/freed-dev-llc/turing-ansible-cluster)** — K3s clusters on RK1 with NPU support via Terraform + Ansible
- **[turing-rk1-cluster](https://github.com/freed-dev-llc/turing-rk1-cluster)** — Reference architecture for 4-node bare-metal K8s
