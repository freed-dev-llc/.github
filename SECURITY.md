# Security Policy

## Reporting a Vulnerability

If you discover a security vulnerability, please report it responsibly.

**Do NOT open a public issue.**

Email: [security@freed.dev](mailto:security@freed.dev)

You should receive a response within 48 hours. We will work with you to understand and resolve the issue before any public disclosure.

## Supported Versions

Only the latest release of each project is supported with security updates.
